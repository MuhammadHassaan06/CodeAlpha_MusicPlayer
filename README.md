# HA MUSIC - Web Based Music Player

HA MUSIC is a sleek, modern, and responsive web-based audio player built to provide a seamless music streaming experience directly in the browser. It features a clean user interface with fully synchronized lyrics and dynamic progress tracking.

## 🎵 Features
* **Synchronized Lyrics:** View real-time lyrics that sync perfectly with the current audio playback.
* **Interactive Progress Bar:** Track your song's progress and skip to any part of the track effortlessly.
* **Playlist Management:** Dynamic playlist functionality to browse and switch between multiple tracks.
* **Modern UI/UX:** Fully responsive design built with clean layouts and smooth transitions that look great on desktop and mobile screens.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Backend/Runtime:** Node.js, Express.js
* **Package Management:** npm (Node Package Manager)
